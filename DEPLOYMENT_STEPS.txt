CPANEL DEPLOYMENT STEPS
=======================

1. LOGIN TO CPANEL
   - Access your cPanel account
   - Find "File Manager"

2. UPLOAD FILES
   - Go to public_html folder
   - Upload all PHP files and folders
   - Create "uploads" folder if it doesn't exist
   - Set uploads folder permissions to 755

3. CREATE DATABASE
   - Go to "MySQL Databases" in cPanel
   - Create new database (note the full name)
   - Create database user
   - Add user to database with ALL PRIVILEGES
   - Note: database name, username, password

4. UPDATE db.php
   - Open db.php in File Manager
   - Update with your cPanel database credentials:
     $host = "localhost";
     $user = "your_db_username";
     $pass = "your_db_password";
     $dbname = "your_db_name";

5. CREATE TABLES
   - Go to "phpMyAdmin" in cPanel
   - Select your database
   - Click "SQL" tab
   - Copy/paste entire tables.sql file
   - Click "Go" to execute

6. ADD SAMPLE DATA (Optional)
   - In phpMyAdmin, click "SQL" tab
   - Run prodandcat.sql (adds products and categories)
   - Run flower_education_sample_data.sql (adds education content)

7. CREATE ADMIN ACCOUNT
   - Option 1: Use admin/dab.php (delete after use)
   - Option 2: Insert directly in phpMyAdmin employees table

8. TEST
   - Visit your domain/login.php
   - Test signup, login, cart, checkout

IMPORTANT FILES TO UPLOAD:
- All .php files
- admin folder (with all files)
- uploads folder (create it, set to 755)

DATABASE INFO NEEDED:
- Database name (usually username_databasename)
- Database username
- Database password
- Host (usually localhost)
