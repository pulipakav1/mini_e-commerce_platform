# Flower Education Content Guide

## Content for Flower Education Section

Project requirement: "Have a section of the site that educates the public about the flower. All content on the website that is displayed must be in the database"

Focus: Tulips only

## 4 Topics Included

1. Tulips: An Introduction - Basic info about tulips
2. Tulip Colors and Their Meanings - What each color means
3. The History and Origin of Tulips - Where they came from
4. How to Plant and Grow Tulips - How to grow them

Each topic includes:
- Description
- Care instructions
- Origin
- Symbolism

## Notes

Sample data file includes 4 entries about tulips. Run the SQL file to add the data to your database.

The education page displays all content from the flower_education table.
